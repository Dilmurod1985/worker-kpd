📊 Система учета КПД сотрудников (Worker-KPD)
Веб-приложение для автоматизации расчета эффективности сотрудников. Разработано специально для Dilmurod Group.

🚀 Основные функции
База сотрудников: Управление списком (ФИО, категория, ID).

Массовый ввод: Копирование данных прямо из Excel/Таблиц.

Авто-табель: Мгновенный расчет процента выполнения нормы в зависимости от категории.

Надежность: Подключена база данных PostgreSQL с защитой SSL.

🛠 Технический стек
Язык: Python 3.11

Фреймворк: Flask

База данных: SQLAlchemy (PostgreSQL на Render)

Верстка: HTML5 + CSS (Bootstrap)

📋 Инструкция для оператора
1. Добавление сотрудников
Перейдите во вкладку "Сотрудники".

Вставьте список в формате: ID [TAB] ФИО [TAB] Категория.

Нажмите "Сохранить".

2. Ежедневный ввод данных
Выберите дату и смену.

В поле ввода вставьте данные из рабочего журнала:
ID_сотрудника Отдел КПД Калибр Штук

Система сама найдет сотрудника по ID и применит его норму.

3. Просмотр результатов
Во вкладке "Табель" записи сортируются автоматически: те, у кого самый низкий процент выполнения, оказываются вверху списка для быстрого контроля.

⚙️ Настройка для разработчика (Deploy на Render)
Для корректной работы на Render.com необходимы следующие переменные окружения (Environment Variables):

DATABASE_URL: Ссылка на внешнюю базу PostgreSQL.

PYTHON_VERSION: 3.11.10

SECRET_KEY: Ваш уникальный ключ безопасности.
